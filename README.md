# Cal using React, Redux & TypeScript

A simple Calendar Reminder app


## Setup

```
$ npm install or yarn
```

## Running

```
$ npm start or yarn start
```

## Directory Breakdown

```
// core - contains main app components

// calendar - all calendar related logic/components

// store - redux store

// assets - all app svg images
```
